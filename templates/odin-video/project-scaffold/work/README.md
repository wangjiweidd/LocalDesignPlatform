# Work

This folder contains intermediate production files:

- `script.md`: editable master script with paragraph IDs.
- `odin-video-spec.md`: upstream creative spec for point of view, designer aesthetic direction, materials, and HyperFrames storyboard.
- `voiceover-minimax.txt`: Minimax-ready voiceover text generated from the script.
- `html/`: HyperFrames animation preview.
- `_codex/`: agent-only temporary files, notes, timing plans, and check frames.

Do not put final MP4 files here. The confirmed export belongs in the project root as `final.mp4`.
