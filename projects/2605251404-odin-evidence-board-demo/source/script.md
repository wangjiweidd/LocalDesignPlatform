# Script

## 标题

Odin 证据板模板 Demo

## 口播脚本

如果 AI 项目只放工具截图，面试官很难判断你到底做了什么。

这类项目要先把材料摆出来：目标是什么，限制在哪里，哪些方案被你放弃了。

真正能加分的，不是工具过程本身，而是你能不能解释清楚判断依据。

所以作品集里可以按三段来讲：旧问题、你的判断、最后的结果。

以后每期视频都可以围绕一份材料展开，画面负责指证据，字幕负责讲判断。

## 屏幕与字幕

1. 屏幕：材料先出现  
   字幕：AI 项目只放工具截图，很难看出你做了什么

2. 屏幕：圈出问题  
   字幕：面试官要看的，是你的判断依据

3. 屏幕：旧问题 / 判断 / 结果  
   字幕：作品集可以按这三段来讲

4. 屏幕：标出改法  
   字幕：目标、限制、取舍，要比工具截图更有用

5. 屏幕：模板方向  
   字幕：画面负责指证据，字幕负责讲判断
