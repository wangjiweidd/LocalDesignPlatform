# 发布正文

设计师面试里，很多人不是没能力，是不会把能力讲成面试官能立刻听懂的价值。

"会装"不是撒谎，也不是教你吹牛。它更像一种专业包装：把你已经做过的事，换成更有判断力、更容易被追问、也更像高级设计师的表达方式。

这条视频讲 5 个最实用的技巧：

1. 不要只说"我做了个改版"，要说清你把哪个指标往前推了多少。
2. 不要只报结果，要让面试官听见你为什么这么判断。
3. 专业术语可以用，但每个词你都要解释得出来。
4. 把草图、白板、迭代记录拿出来，比空讲方法论更有说服力。
5. 主动引导面试官先看哪个项目，本身就是成熟度。

但红线也很清楚：

- 不能编不存在的数据和经历
- 不能把不会的术语硬套到自己身上

能力是基础，包装是放大器。真正加分的"会装"，最后一定能回到真实项目证据。
