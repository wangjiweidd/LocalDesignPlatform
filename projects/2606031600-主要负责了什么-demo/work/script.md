# 01
“你在这个项目里主要负责了什么？”面试讲得越认真，反而越容易被刷掉。尤其是当面试官问到这个问题时，90% 的视觉设计师回答都是错的。

# 02
这道题的关键，其实根本不在于“细节”，而在于“逻辑”。面试官想听的，是基于这样一套逻辑的完整叙事。

# 03
先看活动设计项目。像大促、节日营销、周年庆、运营活动主视觉这些项目类型，面试官真正想听的是：你负责的核心工作、定义视觉语言和信息节奏的方法、以及多渠道上线后的点击和转化表现。

# 04
这段完整的回复话术，你只需要替换活动主题、风格关键词、渠道和结果，就能直接用在面试里。

# 05
再看品牌升级类项目。最常见的错误讲法是：“我做了新 Logo，还做了官网、海报、周边这些应用物料。”视角太低，太偏执行。

# 06
你的回复需要涵盖：旧版本存在的问题，推导新视觉体系的探索和思考，覆盖了哪些关键触点，以及识别度和专业感上的市场反馈。

# 07
不管是什么类型的视觉设计师，只要你能基于这套逻辑回答，你就是一个有决策能力的视觉设计师。如果还能讲出叙事感，offer 就真的一步之遥了。
